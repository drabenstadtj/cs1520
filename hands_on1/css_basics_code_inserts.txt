3. Difference between HTML and CSS
    * HTML provides content and general structure of the page
    * CSS provides styling / content formatting (see example in the W3Schools)

5. Starting a new hands-on project
5.1 Creating a directory to hold your new project

    02 - CSS

5.2 in a terminal window, do the following:

    cd path/to/the/project/directory
    code .     (to run VS Code app in that directory)

5.3 Creating a html skeleton file
5.3.1 Creating an HTML file:
    In the VS Code Explorer area, create a new file, css_basics.html

5.3.2 Creating the minimum HTML content 

    In the new file edit area, type "doc" and ENTER

5.3.3 Displaying your page in the browser
5.3.3.1 Adding the "Live Server" extension by Ritwick Dey into your VS Code

    Go to Extensions and search "Live Server" by Ritwick Dey and install
    Install extension

5.3.3.2 Displaying the page in the browser

    In the html file, right-click and select option "open with live server"

5.2 Creating a CSS file
5.2.1 Creating the file
    From the Explorer area, add a new file, named css_basics.css

5.2.2 Linking the file to the HTML file

    <link rel="stylesheet" href="css_basics.css">

6. changing features of the same html tag throughout the page
    In the html file add this lines

    <h1> Changing all the H2 Headings </h1>
    <h2> First heading </h2>
    <h2> Second heading </h2>
    <h2> Third heading </h2>
    <h2> Fourth heading </h2>

    In the css file add the following lines:

    h2 {
        color:blue;
    }

7. changing the features of a selected group of html tags
    in the html file:

    <hr>
    <h1>changing only few paragraphs, not all</h1> 
    <p class="test_ok"> Test 01: Ok </p>
    <p class="test_fail"> Test 02: Fail </p>
    <p class="test_ok"> Test 03: Ok </p>
    <p class="test_ok"> Test 04: Ok </p>
    <p class="test_fail"> Test 05: Fail </p>

    in the css file:

    .test_ok {
        color:green;
    }

    .test_fail {
        color:red;
        font-weight: bold;
    }

10. Changing a block command to an inline command
    in the html file:

    <hr>
    <h1>Changing a block command to an inline command</h1>
    <a class="inline_paragraph" href="http://www.google.com"> Google link </a>
    <p class="inline_paragraph"> this is a paragraph </p>
    <a class="inline_paragraph" href="http://www.amazon.com"> Amazon link </a>

    in the css file:

    .inline_paragraph {
        float:left;
        margin: 10px;
    }

11. creating a vertical navigation bar from an ul
    in the html file:

    <hr>
    <h1> creating a vertical navigation bar from an ul </h1>
    <ul id="vertical_navigation_1">
        <li> 
            <a href="">
                <h3>home</h3>
            </a>
        </li>
        <li> 
            <a href="">
                <h3> history </h3>
            </a>
        </li>
        <li> 
            <a href="">
                <h3> play list </h3>
            </a> 
        </li>
        <li> 
            <a href=""> 
                <h3>series</h3>
            </a> 
        </li>
    </ul>

    in the css file:

    hr {
        clear:both;
    }

    #vertical_navigation_1 {
        list-style-type: none;
        padding: 0px;
    }

    #vertical_navigation_1 a {
        text-decoration: none;
        color: inherit;
    }

    #vertical_navigation_1 h3 {
        margin: 5px;
    }
